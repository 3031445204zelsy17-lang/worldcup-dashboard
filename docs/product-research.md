## 竞品格局

### 1. FiveThirtyEight (SPI 模型) — 方法论标杆
- **核心**：Soccer Power Index (SPI) = f(进攻评级, 防守评级)
- **方法**：基于 Elo 但用三轴表现数据：① 调整进球 ② 基于射门 xG ③ 非射门 xG
- **状态**：已停更（2024），但方法论是行业模板
- **参考价值**：★★★★★ SPI 教科书

### 2. The Analyst / Opta Supercomputer — 锦标赛预测标杆
- **核心**：2026 WC 冠军概率（西班牙 16% / 法国 12.76% / 英格兰 11.36%）
- **方法**：团队 Elo + 大规模蒙特卡洛模拟（逐场模拟淘汰赛路径）
- **参考价值**：★★★★★ 锦标赛晋级概率展示方式直接对标

### 3. Forebet — 大规模联赛预测
- **核心**：850+ 联赛，胜/平/负概率、预测比分
- **方法**：数学算法 + ML，基于 2000 年起的历史数据
- **状态**：官网被 Cloudflare 拦截，无法直接抓取
- **参考价值**：★★★★ 覆盖广度

### 4. TheDatabetics — 分层建模（新兴）
- **核心**：三层模型架构：① 比赛层 ② 团队/事件层 ③ 球员层
- **理念**：不同市场用不同模型；强调"概率区间而非确定结果"
- **参考价值**：★★★★ "语境因子"（对手质量、赛程、气候）很贴合 2026 WC

### 5. SofaScore / PredictZ / eScored — 消费级
- **SofaScore**：实时动量图 (momentum graph) 是实时 UX 亮点
- **eScored**：AI + 社区 + 实时通知（自夸倾向，谨慎）
- **参考价值**：★★★ SofaScore 动量图是好例子

## 设计模式共识

### 实时胜率怎么展示？
**主流：胜率曲线图 (win probability chart)**
- ESPN：横轴=比赛进程，纵轴=两队胜率，关键事件标在转折点
- SofaScore：动量图，显示哪队当前占优
- 大部分产品：赛前/锦标赛用百分比条

**结论**：比赛中 → 曲线图（体现动态）；赛前/锦标赛 → 百分比条（体现对比）

### 锦标赛晋级概率用什么图表？
- 概率排行榜（ranked bar chart）：按冠军/四强/出线概率排序
- 晋级路径树（bracket）：Opta 在 WC 用，逐轮晋级概率
- 分组表 + 概率列：出线概率附加到小组积分旁

### 信息架构（第一眼看什么？）
行业共识：层级递进
1. **顶层**：关注队的最新概率（一眼可见的核心数字）
2. **次层**：对手对比、关键驱动因素（xG、状态、伤停）
3. **深层**：方法论说明、历史准确率、模拟细节

> TheDatabetics 总结："不是告诉你发生了什么，而是解释为什么某些结果更可能"

### 常见功能模块
| 模块 | 说明 | MVP 需要？ |
|------|------|-----------|
| 胜率曲线图 | 实时比赛进程 | ✅ 核心 |
| 锦标赛概率榜 | 冠军/四强/出线 | ✅ 核心 |
| 球队 Elo/SPI 评级卡 | 实力量化 | ✅ |
| 预测比分 | 具体比分概率 | 可选 |
| 关键事件标注 | 进球/红牌标在曲线上 | ✅ 高价值 |
| 方法论透明页 | 解释模型原理 | ✅ 建立信任 |
| 历史准确率 | 模型回测 | ✅ 建立信任 |
| 伤停/阵容影响 | 实时输入 | 可选 |

## 六大差异化机会

| # | 机会 | 说明 |
|---|------|------|
| 1 | **实时性严重不足** | 绝大多数产品是赛前静态或赛后更新。面向 WC 的实时 Web 仪表盘几乎空白。**最大机会窗口** |
| 2 | **事件驱动"为什么"解释缺失** | 大部分只给数字，不解释为什么变。可在转折点自动标注"此刻发生了什么" |
| 3 | **方法论透明度两极分化** | 538 透明但死；Opta 黑箱；Forebet 不可访问。"透明的活模型"是空白 |
| 4 | **单设备/单场景聚焦不足** | 现有多为大而全。可做单一场景极致：WC 期间、移动优先、秒级更新 |
| 5 | **置信度/不确定性可视化** | 无产品把不确定性本身可视化（阴影带、置信区间） |
| 6 | **地理/旅行/气候因子** | 2026 跨美加墨特殊性，无人产品化 |

## 避坑

- ❌ eScored 式"AI 驱动 + 社区"自吹需谨慎
- ❌ 不要做成博彩导向站点（定位"分析工具"，规避合规）
- ❌ 不要追求"最准"（那是博彩公司的赛道），卖点是实时+透明+可解释

---

## 对我们设计的启发

1. **核心展示**：胜率曲线图（带事件标注）+ 锦标赛概率榜
2. **信息层级**：核心数字 → 驱动因素 → 方法论
3. **差异化落点**：实时性 + 事件解释 + 不确定性可视化
4. **信任建设**：方法论透明页 + 历史准确率展示
